tabla de algunas etiquetas en css

Propiedad CSS Descripción Ejemplo
color Define el color del texto de un elemento. p { color: #11111; }
background-color Establece el color de fondo de un elemento. div { background-color: lightblue; }
font-size Define el tamaño de la fuente (texto). h1 { font-size: 2.5em; } /* O 32px, etc. */
font-family Especifica la fuente (tipo de letra) para el texto. body { font-family: Arial, sans-serif; }
font-weight Establece el grosor (peso) de la fuente (normal, bold, números). .destacado { font-weight: bold; }
text-align Alinea el texto horizontalmente dentro de su contenedor (left, right, center). .titulo { text-align: center; }
margin Define el espacio exterior alrededor de un elemento (fuera del borde). img { margin: 15px; } /* Espacio en todos los lados */
padding Define el espacio interior de un elemento (entre el contenido y el borde). button { padding: 10px 20px; } /* 10px arriba/abajo, 20px izq/der */
